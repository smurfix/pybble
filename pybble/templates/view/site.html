Pybble

This is a master Pybble site. Its content is auto-generated from the source code. You should not register here, as there's nothing you can do anyway.

Pybble is a small, extensible, object-oriented web content publisher. Its main point is that any object in the system can be hooked to any other object. Pybble has a very flexible permission model.
Pybble is currently too inefficient to be used for larger sites. That will change.
For further information, see the Documentation pages.

You can download the current Pybble version at the main author's GIT site.
